# Sombras-del-avismo
Proyecto de magics para el jueves 23 de abril.

## Descripción
Juego de cartas implementado en Java con programación orientada a objetos y una interfaz gráfica perfecta utilizando Swing.

## Características
- Clases OOP: Card, CreatureCard, SpellCard, Player, Game
- Interfaz gráfica con Swing
- Manejo de turnos
- Dibujar cartas
- Mano del jugador

## Cómo ejecutar
1. Compilar: `javac -cp . src/main/java/com/sombrasdelavismo/*.java`
2. Ejecutar: `java -cp . com.sombrasdelavismo.Main`

## Cartas incluidas
- Dragon (Criatura, Costo 5, 4/4)
- Fireball (Hechizo, Costo 3, Daño 3)
- Goblin (Criatura, Costo 1, 1/1)
- Lightning Bolt (Hechizo, Costo 1, Daño 3)
